# Projeto ARANDU, a assistente para professores
## Justificativa
Dificuldades dos professores em informática.
## Hipótese
Melhorar a eficiência nas aulas com nossa assistente.
## Objetivo Geral
Desenvolver uma assistente semelhante à Alexa para ser o primeiro nível de suporte.
## Objetivos Específicos
- Entender níveis de suporte
- Programar um assistente em C++
- Integrar assistente em um projeto físico
## Metodologia
- Estudar sistema
- Esquematizar projeto
- Verificar problemas na escola com pesquisa empírica
- Prototipar aparelho
- Testar aparelho
- Documentar resultados
## Resultado desejado
Esperamos uma autonomia maior para os educadores e otimizar seu tempo em aula.
